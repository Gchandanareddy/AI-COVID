# COVID Symptoms Causal Analysis (Python)

This project implements causal analysis of COVID symptoms entirely in Python.

## Features
- LASSO / Logistic LASSO for network structure
- DAG construction using NetworkX
- CPT estimation (replacement for Netica)
- COVID probability prediction
- SHAP explainability

## Files
- covid_analysis.ipynb → main notebook
- covid_lasso_results.csv → regression results
- covid_network_edges.csv → network edges

## Dataset
http://openonlinecourses.com/causalanalysis/CovidSymptoms.html

## How to run
1. Update dataset path
2. Run notebook top-to-bottom
